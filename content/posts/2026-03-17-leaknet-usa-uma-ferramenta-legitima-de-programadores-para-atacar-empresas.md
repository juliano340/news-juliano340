# LeakNet usa ferramenta legítima para atacar empresas
Grupo criminoso instala ambiente de execução JavaScript e usa confiança do sistema para executar malware sem deixar rastros.

## Resumo em 3 bullets
- LeakNet usa ClickFix para enganar vítimas e executar comandos maliciosos
- Grupo instala Deno, ferramenta legítima, para rodar código na memória
- Estratégia BYOR deixa mínimos rastros e evita detecção por antivírus

## Contexto
LeakNet é grupo criminoso especializado em ransomware, ativo desde 2024, com média de três vítimas por mês.

## Insights e implicacoes
Uso de ferramentas legítimas como Deno representa nova tendência em ataques cibernéticos, dificultando detecção tradicional.
