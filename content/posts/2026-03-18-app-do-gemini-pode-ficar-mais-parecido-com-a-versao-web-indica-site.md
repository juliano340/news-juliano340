# App do Gemini pode ficar mais parecido com a versão web, indica site
O aplicativo do Gemini para Android pode mudar em breve. Informações do site Android Authority indicam que o app passa por testes de uma nova interface mais similar à versão web, além de uma possível aba chamada “Discover”.

## Resumo em 3 bullets
- App do Gemini pode ganhar interface mais parecida com versão web
- Nova aba “Discover” pode trazer recomendações e prompts da comunidade
- Menu lateral será reorganizado para incluir nova aba e acesso rápido a configurações

## Contexto
O Gemini é o assistente de inteligência artificial do Google, disponível em diferentes plataformas. Atualmente, as versões para web e mobile têm interfaces distintas, mas mudanças em desenvolvimento sugerem uma padronização entre os sistemas.
