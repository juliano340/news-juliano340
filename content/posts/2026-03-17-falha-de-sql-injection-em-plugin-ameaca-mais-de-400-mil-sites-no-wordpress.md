Vulnerabilidade crítica no plugin Ally pode permitir extração de dados de sites WordPress.

## Resumo em 3 bullets
- Plugin Ally tem falha crítica de SQL injection
- Mais de 400 mil sites WordPress estão vulneráveis
- Falha permite extração de dados confidenciais

## Contexto
Plugin de acessibilidade Ally, usado por milhares de sites WordPress, apresenta vulnerabilidade crítica que permite a criminosos extrair dados sensíveis dos bancos de dados afetados.
