# Uber: código para confirmar viagens passa a ser fixo
Uber altera código de confirmação para combinação fixa, gerada aleatoriamente no primeiro uso.

## Resumo em 3 bullets
- Código fixo pode ser decorado para iniciar corridas futuras.
- Recurso é opcional e pode ser ativado em horários específicos.
- Sequências fáceis como 1234 ou 0000 são recusadas pelo aplicativo.

## Contexto
A Uber está mudando seu sistema de código de confirmação de viagens. Até agora, o código era aleatório e mudava a cada corrida. Agora, será gerado aleatoriamente apenas no primeiro uso e permanecerá o mesmo para as próximas viagens, com opção de alteração pelo usuário.

## Insights e implicacoes
A mudança visa facilitar o início de corridas em situações onde o tempo é precioso, como quando a bateria do celular está acabando ou em locais com grande aglomeração de pessoas. O recurso continua sendo opcional e pode ser ativado por usuários ou motoristas para períodos específicos.

## O que fazer agora
- Configure o código fixo se preferir não digitar um novo código a cada viagem.
- Escolha uma combinação memorável mas segura, evitando sequências óbvias.
- Ative o recurso apenas nos horários em que achar necessário para maior segurança.

## O que vale acompanhar
- Confirme se o código fixo está ativado antes de solicitar uma corrida.
- Verifique se o motorista está solicitando o código nos períodos configurados.
- Monitore se o novo sistema está funcionando corretamente nas primeiras viagens.
