# Amazon compra Globalstar para competir com Starlink, de Musk
A Amazon anunciou a aquisição da empresa de satélites Globalstar por US$ 11,57 bilhões, reforçando seu projeto de internet via satélite.

## Resumo em 3 bullets
- Amazon compra Globalstar por US$ 11,57 bilhões para competir com Starlink
- Globalstar opera satélites usados no SOS de Emergência da Apple
- Amazon planeja lançar 3.200 satélites até 2029

## Contexto
A Amazon busca fortalecer seu projeto de satélites, chamado Amazon Leo, para competir com a líder Starlink, de Elon Musk, que já atende milhões de usuários globalmente.
