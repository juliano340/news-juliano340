# Windows 11 terá teste de velocidade de internet nativo
Microsoft anuncia funcionalidade que medirá desempenho de conexões Ethernet, Wi-Fi e dados móveis diretamente no sistema operacional.

## Resumo em 3 bullets
- Teste de velocidade nativo será acessado pelo ícone de rede na bandeja do sistema
- Funcionalidade suportará conexões Ethernet, Wi-Fi e dados móveis
- Recurso já está disponível para Windows Insiders no canal Release Preview

## Contexto
A Microsoft está integrando um teste de velocidade de internet diretamente ao Windows 11, eliminando a necessidade de acessar sites ou aplicativos externos para medir o desempenho da rede. A funcionalidade será acessada clicando com o botão direito no ícone de rede da bandeja do sistema e abrirá no navegador padrão do usuário.

## Insights e implicacoes
Esta novidade reflete a tendência de sistemas operacionais se tornarem mais autossuficientes em diagnósticos básicos, melhorando a experiência do usuário ao centralizar ferramentas essenciais. A integração nativa pode tornar o processo de solução de problemas de conectividade mais rápido e intuitivo para usuários menos técnicos.
