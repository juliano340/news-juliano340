# Firefox encerra suporte para Windows 7, 8 e 8.1 em março de 2026
Mozilla confirmou que não haverá mais atualizações do Firefox para sistemas operacionais descontinuados pela Microsoft.

## Resumo em 3 bullets
- Firefox 115 ESR será a última versão compatível com Windows 7, 8 e 8.1
- Atualizações de segurança terminarão em fevereiro de 2026
- Mozilla recomenda migrar para Windows 10, 11 ou distribuições Linux

## Contexto
A Microsoft encerrou suporte oficial aos Windows 7, 8 e 8.1 em janeiro de 2023, deixando estes sistemas vulneráveis a ameaças de segurança. O Firefox era uma das poucas opções de navegador seguro ainda disponível para estas versões.

## Insights e implicacoes
Com o fim do suporte, usuários destes sistemas operacionais ficarão expostos a riscos de segurança, pois o navegador deixará de receber correções de vulnerabilidades. A decisão reflete a dificuldade de manter compatibilidade com sistemas desatualizados.

## O que fazer agora
- Verifique qual versão do Windows está instalada no seu computador
- Considere atualizar para Windows 10, 11 ou instalar uma distribuição Linux
- Faça backup de seus dados importantes antes de migrar de sistema

- Anúncios de novas datas de fim de suporte de softwares
- Recomendações de segurança para sistemas desatualizados
- Alternativas de sistemas operacionais leves para computadores antigos
