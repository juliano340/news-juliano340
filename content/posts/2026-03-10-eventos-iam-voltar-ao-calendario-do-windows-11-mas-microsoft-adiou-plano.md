# Microsoft adia recurso de eventos no calendário do Windows 11
Microsoft adiou adição de eventos ao calendário do Windows 11, inicialmente prevista para testes no final de 2025; função ainda está nos planos, mas data de liberação da prévia não foi especificada; recurso já existia no Windows 10, mas foi retirado do Windows 11 pela Microsoft.

## Resumo em 3 bullets
- Microsoft adiou testes do recurso de eventos no calendário do Windows 11
- Funcionalidade já existia no Windows 10, mas foi removida do Windows 11
- Desenvolvimento em WebView pode ter causado problemas de desempenho

## Contexto
O calendário flutuante do Windows 11 atualmente serve apenas para consulta de datas, enquanto o Windows 10 permitia adicionar eventos e lembretes diretamente. A Microsoft havia prometido trazer essa funcionalidade de volta em testes no final de 2025, mas adiou sem especificar nova data.

## Insights e implicacoes
A demora sugere que a Microsoft pode estar redesenhando a interface usando componentes nativos do Windows em vez de WebView, o que exigiria mais tempo de desenvolvimento mas poderia resultar em melhor desempenho.

## O que fazer agora
- Monitore atualizações do programa Windows Insider
- Considere usar aplicativos de calendário de terceiros como alternativa
- Prepare-se para possível mudança na interface do calendário flutuante

## O que vale acompanhar
