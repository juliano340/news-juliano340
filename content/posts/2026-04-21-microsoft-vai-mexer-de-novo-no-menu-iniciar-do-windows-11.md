# Microsoft vai mexer de novo no Menu Iniciar do Windows 11
Microsoft prepara nova atualização para Menu Iniciar do Windows 11 com base no WinUI 3, trazendo mais opções de personalização e melhor desempenho.

## Resumo em 3 bullets
- Menu Iniciar será refeito com base no WinUI 3 para melhorar desempenho e padronização
- Usuários poderão ativar ou desativar segmentos específicos do Menu Iniciar
- Opção de escolher entre layout pequeno ou grande do componente

## Contexto
Após recente reformulação do Menu Iniciar em outubro de 2025, Microsoft planeja nova atualização para tornar o componente mais personalizável e eficiente. A mudança está sendo desenvolvida com base no WinUI 3, estrutura moderna para interfaces de usuário no Windows.

## Insights e implicacoes
A Microsoft busca equilibrar personalização e desempenho, permitindo que usuários ajustem o Menu Iniciar conforme suas preferências sem comprometer a velocidade de abertura, mesmo em situações de alta carga de processamento.
