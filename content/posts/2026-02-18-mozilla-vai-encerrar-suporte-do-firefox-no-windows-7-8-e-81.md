# Mozilla vai encerrar suporte do Firefox no Windows 7, 8 e 8.1
A partir de fevereiro de 2026, o Firefox não receberá mais atualizações para Windows 7, 8 e 8.1, deixando usuários sem correções de segurança.

## Resumo em 3 bullets
- Firefox 115 foi a última versão compatível com Windows 7, 8 e 8.1
- Usuários perderão atualizações de segurança e novos recursos
- Mozilla cita custos operacionais e baixa base de usuários como motivos

## Contexto
A Mozilla está descontinuando o suporte do Firefox em sistemas Windows antigos devido a custos operacionais e baixa adoção. A partir de fevereiro de 2026, usuários dessas plataformas não receberão mais atualizações, incluindo patches de segurança. A última versão compatível foi o Firefox 115, lançado em julho de 2023.

## Insights e implicacoes
A decisão reflete a tendência do mercado de tecnologia de abandonar sistemas legados. Usuários afetados devem considerar migrar para sistemas operacionais mais modernos para manter a segurança digital.

## O que fazer agora
- Verifique qual versão do Windows está usando
- Faça backup de dados importantes
- Planeje a migração para Windows 11 ou Linux
- Considere atualizar o hardware se necessário
