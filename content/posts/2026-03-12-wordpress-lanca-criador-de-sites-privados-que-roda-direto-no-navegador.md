# WordPress lança criador de sites privados que roda direto no navegador
Nova ferramenta é direcionada a blogs pessoais (imagem: reprodução)

## Resumo em 3 bullets
- WordPress lança myWordPressnet para criar sites privados no navegador
- Não precisa de conta, hospedagem ou domínio para começar
- Dados são armazenados localmente, sem envio para nuvem

## Contexto
A ferramenta myWordPressnet elimina as etapas tradicionais de configuração de uma página. O usuário não precisa criar uma conta, contratar um plano de hospedagem ou registrar um domínio para começar a utilizar o sistema. Diferente das páginas convencionais, os sites criados nesta ferramenta são privados por padrão e não são otimizados para receber tráfego, aparecerem em buscas ou serem apresentados ao público.

## Insights e implicacoes
A organização posiciona o recurso como um espaço de trabalho local para a criação de rascunhos, anotações e para testes de plugins e temas. A infraestrutura do serviço baseia-se no WordPress Playground, a mesma tecnologia que já era utilizada para gerar demonstrações temporárias da plataforma. Agora, no entanto, o ambiente ganha um caráter permanente e pessoal.

## O que fazer agora
- Acessar my.wordpress.net para testar a ferramenta
- Explorar os aplicativos disponíveis como Leitor de RSS e CRM Pessoal
- Fazer backup dos projetos regularmente para evitar perda de dados

## O que vale acompanhar
- Limitação de 100 MB de armazenamento inicial
- Tempo de carregamento no primeiro acesso
- Separação de dados entre diferentes dispositivos
