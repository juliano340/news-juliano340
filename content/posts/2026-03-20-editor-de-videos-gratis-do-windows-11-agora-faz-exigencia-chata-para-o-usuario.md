# Clipchamp agora exige salvar projetos no OneDrive
Microsoft implementa mudança que afeta todos os usuários do editor de vídeos gratuito no Windows 11.

## Resumo em 3 bullets
- Projetos do Clipchamp agora precisam ser salvos no OneDrive para edição
- Versão gratuita tem apenas 5 GB de armazenamento na nuvem
- Contas pagas do Clipchamp também foram afetadas pela mudança

## Contexto
O Clipchamp é um editor de vídeos integrado ao Windows 11, oferecendo ferramentas de edição gratuitas para usuários. Até recentemente, permitia edição totalmente local sem exigir conexão com nuvem.

## Insights e implicacoes
Mudança representa estratégia da Microsoft para aumentar adesão ao OneDrive e aos planos Microsoft 365, limitando funcionalidades da versão gratuita.

## O que fazer agora
- Verificar espaço disponível no OneDrive
- Considerar opções de armazenamento alternativas
- Avaliar necessidade de assinatura Microsoft 365
- Testar editores de vídeo alternativos no Windows

## O que vale acompanhar
- Anúncios de novas funcionalidades pagas
- Reações da comunidade sobre privacidade
- Lançamentos de editores concorrentes
- Mudanças nas políticas de armazenamento
