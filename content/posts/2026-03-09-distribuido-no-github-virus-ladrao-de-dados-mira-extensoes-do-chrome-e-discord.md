# Campanha no GitHub distribui malware que rouba dados do Chrome e Discord
Criminosos usam repositórios falsos no GitHub para distribuir o BoryptGrab, um malware que rouba dados de navegadores, extensões e tokens de autenticação.

## Resumo em 3 bullets
- Malware BoryptGrab é distribuído via repositórios falsos no GitHub
- Ataca dados do Chrome e outras extensões de navegadores
- Foca em carteiras de criptomoedas e tokens do Discord

## Contexto
Ataque sofisticado usa técnicas de SEO e redirecionamento para distribuir malware via GitHub, visando dados sensíveis armazenados em navegadores e extensões.

## Insights e implicacoes
A operação demonstra evolução nas táticas de distribuição de malware, usando plataformas legítimas e técnicas de ofuscação para escapar de detecção.

## O que fazer agora
- Evite baixar software de repositórios desconhecidos no GitHub
- Mantenha antivírus e navegadores sempre atualizados
- Use autenticação de dois fatores em contas importantes

## O que vale acompanhar
- Repositórios falsos oferecendo ferramentas gratuitas
- Downloads redirecionados por múltiplas páginas
- Arquivos ZIP com executáveis desconhecidos
