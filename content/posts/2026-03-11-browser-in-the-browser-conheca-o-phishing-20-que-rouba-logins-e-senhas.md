# Browser-in-the-browser: conheça o 'phishing 2.0' que rouba logins e senhas
Nova técnica de phishing simula janela pop-up dentro do navegador para roubar dados de acesso.

## Resumo em 3 bullets
- Browser-in-the-browser (BitB) simula pop-up dentro de página fraudulenta
- Janela falsa mostra URL legítima para enganar usuário
- Golpe imita login de serviços como Facebook, Google e Microsoft

## Contexto
Phishing tradicional evoluiu para técnica mais convincente que simula janela de login real dentro do navegador, aumentando risco de roubo de dados.

## Insights e implicacoes
Criminosos agora usam desenvolvimento web avançado para criar simulações mais realistas de páginas de login, tornando golpes mais difíceis de identificar.

## O que fazer agora
- Use gerenciador de senhas para verificar URLs
- Habilite autenticação em dois fatores em todas as contas
- Desconfie de janelas pop-up que pedem login
- Verifique endereço real da página antes de inserir dados

## O que vale acompanhar
- Simulações de login em pop-ups falsos
- Avisos por email sobre problemas na conta
- Páginas que imitam serviços populares
- Janelas que mostram URLs editadas
- Formulários que pedem dados sensíveis
