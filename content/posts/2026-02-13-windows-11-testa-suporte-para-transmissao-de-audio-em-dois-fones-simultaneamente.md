## Resumo em 3 bullets
- **Topico 1**: Windows 11 está testando transmissão simultânea de áudio para dois dispositivos via Bluetooth Low Energy
- **Topico 2**: Funcionalidade disponível em prévia para participantes do Windows Insider em PCs Copilot+ com Snapdragon X
- **Topico 3**: Limite de dois dispositivos conectados ao mesmo tempo, configurável nas Configurações Rápidas

## Por que isso importa para devs
A capacidade de transmitir áudio para dois dispositivos simultaneamente pode otimizar workflows colaborativos, como pair programming ou revisões de código compartilhadas, eliminando a necessidade de cabos ou equipamentos adicionais. Para desenvolvedores que trabalham com sistemas multimídia ou aplicações de áudio, essa funcionalidade nativa do Windows 11 pode acelerar testes e validações de compatibilidade com múltiplas saídas de som, especialmente em ambientes de desenvolvimento móvel ou embarcado.

## O que muda na pratica
- Verificar se seus fones de ouvido ou caixas de som suportam Bluetooth Low Energy (BLE) para compatibilidade com o recurso
- Participar do programa Windows Insider para testar a funcionalidade na Build 26220.7051
- Acompanhar atualizações do Windows Insider para saber quando o suporte será expandido para processadores Intel Core Ultra

## Contexto rapido
- Funcionalidade baseada no protocolo Auracast, que permite transmissão de uma fonte para múltiplas saídas
- Disponível inicialmente apenas para PCs Copilot+ equipados com processadores Qualcomm Snapdragon X
- Microsoft não divulgou previsão de lançamento para o público geral
