# Claude Code: vulnerabilidades permitiam acesso remoto e roubo de chaves de API
Pesquisadores da Check Point Research descobriram falhas graves no Claude Code, ferramenta de linha de comando da Anthropic. As vulnerabilidades permitiam execução remota de código, bypass de consentimento do usuário e roubo de credenciais.

## Resumo em 3 bullets
- Hooks maliciosos executavam sem aviso ao inicializar o Claude Code
- Configurações MCP permitiam aprovação automática de servidores externos
- Redirecionamento de tráfego possibilitava roubo de chaves de API

## Contexto
O Claude Code é uma ferramenta de linha de comando da Anthropic que permite aos desenvolvedores usar inteligência artificial diretamente no terminal. Ele usa arquivos de configuração como .claude/settings.json para definir comportamentos e hooks que rodam automaticamente durante seu ciclo de vida.

## Insights e implicacoes
As vulnerabilidades exploravam a confiança em arquivos de configuração, que normalmente não são analisados com o mesmo rigor que código. Atacantes podiam esconder comandos maliciosos em repositórios falsos, comprometendo a cadeia de suprimentos de desenvolvimento.

## O que fazer agora
- Atualize o Claude Code para a versão mais recente
- Revise arquivos de configuração em repositórios desconhecidos
- Desconfie de hooks e MCP servers automáticos em projetos externos

## O que vale acompanhar
- Novas atualizações de segurança do Claude Code
- Mudanças nos protocolos de consentimento do usuário
- Relatórios de incidentes envolvendo supply chain attacks
