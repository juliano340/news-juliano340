# Gemini ganha recurso para criar músicas com IA; veja como funciona
O Google anunciou o Lyria 3, modelo de IA que permite criar músicas no Gemini a partir de comandos simples em diferentes formatos.

## Resumo em 3 bullets
- Lyria 3 não exige letra para gerar músicas
- Permite controle sobre estilo, vocal e ritmo
- Gera faixas de até 30 segundos com capas personalizadas

## Contexto
O Google está expandindo as capacidades do Gemini com um novo modelo de IA musical chamado Lyria 3. Diferente de versões anteriores, não é mais necessário fornecer letra para gerar músicas. O usuário pode descrever gênero, tema ou enviar referências visuais para criar faixas personalizadas. O recurso também integra-se ao YouTube para criação de trilhas para Shorts.

## Insights e implicacoes
A ferramenta representa um passo significativo na democratização da criação musical, permitindo que usuários sem formação musical expressem criatividade através de IA. A inclusão de marca d'água SynthID demonstra preocupação com transparência e direitos autorais.

## O que fazer agora
- Acesse o Gemini em sua versão desktop
- Clique na aba 'Música' para selecionar o modelo Lyria 3
- Experimente criar músicas com diferentes comandos de texto ou referências visuais

## O que vale acompanhar
- Evolução da qualidade das músicas geradas pelo Lyria 3
- Integração com YouTube Shorts para criação de trilhas
- Limites de geração para usuários gratuitos vs assinantes
