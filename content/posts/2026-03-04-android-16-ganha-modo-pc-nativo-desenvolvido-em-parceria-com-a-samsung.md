# Android 16 ganha modo PC nativo, desenvolvido em parceria com a Samsung
O Android 16 QPR3 introduziu um modo PC nativo, disponível apenas para a linha Pixel, que não é vendida no Brasil.

## Resumo em 3 bullets
- Android 16 QPR3 traz modo PC nativo para linha Pixel
- Recurso permite conectar celular a monitor via USB-C
- Interface similar a desktop, com suporte a mouse e teclado Bluetooth

## Contexto
O modo desktop já estava presente na versão beta QPR1 Beta 2, distribuída em junho de 2025. Agora, a disponibilidade é mais ampla. O Android QPR é uma versão trimestral do sistema, liberada para aparelhos da linha Pixel. Posteriormente, essas novidades podem chegar à versão estável do sistema, cabendo a outras fabricantes implementá-las.

## Insights e implicacoes
O modo desktop do Android é bastante aguardado, mas muitos usuários do sistema já contam com esse recurso, graças a iniciativas das próprias fabricantes de celulares. Uma delas é a própria Samsung, com o modo DeX. Outra pioneira nesse tipo de conectividade é a Motorola.

## O que fazer agora
- Verificar se seu aparelho Pixel 8 ou posterior é compatível
- Atualizar para Android 16 QPR3 para acessar o modo PC
- Conectar celular a monitor via USB-C para testar funcionalidade

## O que vale acompanhar
- Lançamento de Android para PCs pela Google
- Implementação do modo PC por outras fabricantes
- Suporte a modo desktop em tablets Pixel
