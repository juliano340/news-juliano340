# Vidar 2.0: vírus usa trapaças 'falsas' para roubar dados
Pesquisadores identificaram campanha global que usa cheats falsos para distribuir malware que rouba dados de jogadores.

## Resumo em 3 bullets
- Vidar Stealer 2.0 mira jogadores com cheats falsos em GitHub e Reddit
- Malware rouba senhas, cookies, tokens e carteiras de criptomoedas
- Criminosos exploram desconfiança de jogadores em alertas de antivírus

## Contexto
Campanha de roubo de dados usa cheats falsos para infectar jogadores com malware Vidar 2.0, que rouba informações sensíveis e pode comprometer redes corporativas.

## Insights e implicacoes
Malware-as-a-Service permite que criminosos aluguem vírus prontos para usar, facilitando ataques em escala. Jogadores jovens são alvos principais, mas qualquer pessoa que misture jogos e trabalho está exposta.

## O que fazer agora
- Baixar programas apenas de fontes oficiais e verificadas
- Manter sistema operacional e antivírus sempre atualizados
- Usar senhas diferentes para cada serviço e ativar verificação em duas etapas

## O que vale acompanhar
- Publicações em subreddits ou servidores Discord oferecendo cheats gratuitos
- Arquivos com nomes de ferramentas de jogos como TempSpoofer.exe ou EzFrags_Private.zip
- Links em imagens que direcionam para páginas no GitHub
