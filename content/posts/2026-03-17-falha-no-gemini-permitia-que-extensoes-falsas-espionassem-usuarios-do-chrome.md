# Vulnerabilidade no Chrome permitia espionagem via Gemini
Pesquisadores descobriram falha no WebView do Chrome que permitia a extensões maliciosas explorar o Gemini e espionar usuários.

## Resumo em 3 bullets
- Falha no WebView permitia injeção de código JavaScript no Gemini
- Extensões maliciosas podiam roubar informações e monitorar atividades
- Google corrigiu problema nas versões 143.0.7499.192 e 143.0.7499.193

## Contexto
O Chrome integrou o Gemini, permitindo interações avançadas, mas também criou vulnerabilidades. A estrutura WebView do navegador, que normalmente impõe restrições, foi explorada para permitir que extensões maliciosas executassem código JavaScript com privilégios elevados. Como o Gemini opera com acesso profundo ao navegador para auxiliar em tarefas, essa falha aumentou significativamente a superfície de ataque.

## Insights e implicacoes
A integração de ferramentas de IA como o Gemini com navegadores e aplicativos pode criar vulnerabilidades críticas, especialmente quando componentes internos como o WebView removem restrições de segurança. Esse caso mostra como privilégios elevados, necessários para funcionalidades avançadas, também ampliam os riscos de exploração por extensões maliciosas.

## O que fazer agora
- Verifique se seu Chrome está atualizado para as versões 143.0.7499.192 (Windows) ou 143.0.7499.193 (macOS)
- Revise e remova extensões desnecessárias do navegador
- Monitore permissões de extensões instaladas
