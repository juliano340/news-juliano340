# 287 extensões do Chrome roubam dados de 37 milhões de usuários
Pesquisa revela que adblockers e assistentes de pesquisa coletam dados privados para vender a terceiros.

## Resumo em 3 bullets
- 287 extensões do Chrome roubam histórico de navegação de 37,4 milhões de usuários
- Add-ons maliciosos vendidos como adblockers ou assistentes de pesquisa
- SimilarWeb é o principal suspeito, afetando até 10,1 milhões de usuários

## Contexto
Extensões do Chrome são ferramentas populares para personalizar a experiência de navegação, mas muitas delas coletam dados dos usuários sem consentimento claro. O problema é agravado por brechas nas políticas de privacidade e exceções legais que permitem a venda dessas informações a terceiros.

## Insights e implicacoes
A tendência de venda de dados a terceiros por ferramentas populares está se tornando comum, aproveitando-se de exceções nas políticas de privacidade. Usuários devem estar cientes de que, se não estão pagando por um produto com dinheiro, estão pagando com seus dados pessoais.

## O que fazer agora
- Revise as extensões instaladas no Chrome e remova as desconhecidas
- Verifique as permissões solicitadas por cada extensão antes de instalá-las
- Prefira extensões de desenvolvedores confiáveis e com boa reputação

## O que vale acompanhar
- Novas extensões maliciosas que possam surgir no Chrome Web Store
- Atualizações nas políticas de privacidade do Chrome e outras plataformas
- Desenvolvimento de ferramentas para detectar e bloquear extensões maliciosas
