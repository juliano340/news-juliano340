# PC inviável: alta das marcas chinesas faz memória RAM bater R$ 3 mil no Brasil
A crise de memórias RAM continua se agravando no Brasil, com preços subindo mesmo para alternativas chinesas que eram vistas como solução.

## Resumo em 3 bullets
- Memórias RAM da marca chinesa KingBank já custam US$ 530 no exterior
- Fabricantes chineses como CXMT podem migrar para memórias HBM mais lucrativas
- Preço de kit 32 GB DDR5 ultrapassa R$ 3 mil no mercado brasileiro

## Contexto
A escassez global de chips de memória RAM está afetando severamente o mercado brasileiro, com preços disparando para níveis que tornam a montagem de PCs cada vez mais inviável. Inicialmente, fabricantes chineses como a CXMT eram vistos como alternativa para driblar a crise, mas até essas opções estão se tornando caras.

## Insights e implicacoes
A migração de fabricantes chineses para produção de memórias HBM para data centers indica que a crise pode se prolongar, já que essas memórias são mais lucrativas e consomem grande parte da capacidade produtiva disponível.
