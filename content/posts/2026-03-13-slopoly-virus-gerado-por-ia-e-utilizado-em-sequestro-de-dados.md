# Slopoly: vírus gerado por IA é utilizado em sequestro de dados
Pesquisadores da IBM X-Force identificaram o malware Slopoly, usado em ataque de ransomware no início de 2026. O código foi provavelmente gerado por inteligência artificial, marcando uma virada no modo como criminosos desenvolvem suas ferramentas.

## Resumo em 3 bullets
- Slopoly é script PowerShell usado como cliente de framework de comando e controle
- Malware foi instalado em pasta que imita diretório legítimo do Windows
- Código apresenta características típicas de geração por IA, como comentários detalhados

O caso representa uma mudança significativa no cenário de ameaças cibernéticas, pois demonstra como inteligência artificial pode ser usada para criar malware com qualidade técnica razoável, reduzindo barreiras para criminosos que não possuem habilidades avançadas de programação.
