Campanha ClickFix explora engenharia social para enganar usuários e instalar vírus no sistema operacional.

## Resumo em 3 bullets
- Microsoft detectou campanha que usa Terminal do Windows para instalar malware
- Criminosos induzem vítima a executar código malicioso em páginas falsas
- Malware Lumma Stealer rouba dados do navegador e credenciais

## Contexto
Ataques de engenharia social continuam evoluindo, usando táticas cada vez mais sofisticadas para enganar usuários. O modelo ClickFix já foi identificado em outras campanhas, mas esta nova abordagem que explora o Terminal do Windows representa uma ameaça específica para usuários da plataforma.

## Insights e implicacoes
Criminosos estão aproveitando a confiança dos usuários em comandos do sistema para distribuir malware. A combinação de instruções passo a passo com páginas falsas de CAPTCHA e alertas de problemas torna o ataque particularmente eficaz.

## O que fazer agora
- Desconfie de páginas que pedem para executar comandos no Terminal do Windows
- Não copie e cole comandos de fontes não verificadas
- Mantenha seu sistema operacional e antivírus sempre atualizados
- Fique atento a páginas falsas de CAPTCHA e alertas suspeitos

## O que vale acompanhar
