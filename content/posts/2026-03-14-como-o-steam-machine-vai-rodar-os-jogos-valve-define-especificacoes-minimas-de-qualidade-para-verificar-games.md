# Como o Steam Machine vai rodar os jogos? Valve define especificações mínimas de qualidade para verificar games
A Valve detalhou como funcionará o sistema de verificação de jogos para o Steam Machine, revelando qual será o alvo mínimo de desempenho para que um título receba o selo de compatibilidade no dispositivo.

## Resumo em 3 bullets
- Valve define 1080p a 30 FPS como padrão mínimo para jogos no Steam Machine
- Sistema de verificação é inspirado no Steam Deck para garantir experiência consistente
- Títulos com selo 'Deck Verified' devem rodar sem problemas no novo dispositivo

## Contexto
A Valve apresentou na Game Developers Conference detalhes sobre o sistema de verificação de jogos para o Steam Machine. A empresa busca padronizar a experiência de uso dentro do novo ecossistema de hardware, seguindo uma lógica semelhante ao sistema já utilizado no Steam Deck.

## Insights e implicacoes
O requisito mínimo de 1080p a 30 FPS não representa o limite do hardware, mas sim uma base para garantir uma experiência consistente aos jogadores. A empresa reforça que muitos jogos poderão ultrapassar esse padrão mínimo, alcançando resoluções mais altas e taxas de quadros superiores dependendo da otimização.
