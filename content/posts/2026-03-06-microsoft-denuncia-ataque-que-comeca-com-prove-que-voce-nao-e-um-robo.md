# Microsoft denuncia ataque que começa com "prove que você não é um robô"
Nova campanha de ClickFix usa CAPTCHA falso para instalar malware Lumma Stealer no Windows.

## Resumo em 3 bullets
- Ataque começa com CAPTCHA falso que pede para abrir terminal do Windows
- Vítima cola comando ofuscado que instala malware disfarçado
- Malware acessa senhas salvas em Chrome e Edge

## Contexto
Ataque combina engenharia social com técnicas avançadas de ofuscação e uso de ferramentas legítimas do sistema.

## Insights e implicacoes
Criminosos aproveitam confiança em CAPTCHAs e ferramentas Microsoft para burlar sistemas de segurança.

## O que fazer agora
- Não cole comandos desconhecidos no terminal do Windows
- Use senhas fortes e únicas para cada serviço
- Mantenha antivírus atualizado e ative proteções adicionais

## O que vale acompanhar
- Páginas que pedem para abrir terminal após CAPTCHA
- Arquivos executáveis com nomes aleatórios
- Atividade suspeita em navegadores Chrome e Edge
