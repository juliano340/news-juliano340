# Google Flow integra Nano Banana e traz novas ferramentas de edição de vídeo
O Google atualizou o Flow com integração do modelo Nano Banana e novos comandos para edição de vídeos, unificando criação de imagens e vídeos em uma única plataforma.

## Resumo em 3 bullets
- Flow agora integra modelo Nano Banana para gerar imagens de alta qualidade
- Nova interface com grid de ativos permite organizar e filtrar mídia
- Ferramentas Whisk e ImageFX serão migradas para biblioteca unificada em março

## Contexto
O Google Flow é uma ferramenta de IA para criação de vídeos cinematográficos a partir de comandos de texto, usando modelos avançados para gerar cenas completas com áudio e personagens. Diferente do Gemini, que é focado em texto, o Flow foi desenvolvido especificamente para criação audiovisual com mais controle criativo.

## Insights e implicacoes
A integração do Nano Banana e as novas ferramentas de edição posicionam o Flow como uma solução mais completa para produção audiovisual, reduzindo a necessidade de alternar entre diferentes plataformas e permitindo maior controle sobre o processo criativo.

## O que fazer agora
- Explore a nova interface com grid de ativos para organizar sua mídia
- Teste os comandos de texto e símbolo @ para edição mais precisa
- Prepare-se para migrar projetos das ferramentas Whisk e ImageFX em março
