# Como criar imagens no Perplexity com o Nano Banana 2
O Perplexity é uma IA multimodal que dá acesso a vários modelos avançados para criação de imagens, incluindo a versão mais recente do Google, o Nano Banana 2.

## Resumo em 3 bullets
- Nano Banana 2 é modelo mais recente do Google para geração de imagens
- Seleção manual do modelo disponível apenas em planos pagos
- IA produz imagens com boa definição e iluminação

## Contexto
O Perplexity permite criar conteúdos visuais a partir de comandos de texto usando diferentes modelos de IA. No entanto, a escolha do modelo específico está restrita aos usuários dos planos Pro e Max.

## Insights e implicacoes
O Nano Banana 2 se destaca pela rapidez no processamento e pela fidelidade aos detalhes descritos no prompt, além de produzir imagens com boa definição, iluminação e textos mais legíveis quando necessário.

## O que fazer agora
- Acesse as configurações da conta no Perplexity
- Vá em 'Preferências' e selecione 'Modelo de geração de imagem'
- Escolha o Nano Banana 2 como padrão
- Envie comandos de texto no chat para criar imagens
